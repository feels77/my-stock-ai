\# AI 주간 점검 설계서



\## 목적



사용자가 매주 1회 Portfolio Doctor를 열고 현재 포트폴리오 상태를 점검한 후 AI에게 분석을 요청할 수 있도록 한다.



\---



\## 사용 시나리오



1\. 사용자가 Portfolio Doctor 실행

2\. 최신 자산 상태 확인

3\. \[주간 자산 점검] 버튼 클릭

4\. 시스템이 분석용 리포트 생성

5\. 사용자가 ChatGPT 또는 Gemini에 붙여넣음

6\. AI가 행동 제안 제공



\---



\## 리포트 포함 항목



\### 1. 포트폴리오 요약



\* 총 평가금액

\* 총 손익

\* 총 수익률



\---



\### 2. 자산군 비중



\* 연금형

\* 성장형

\* 방어형

\* 파킹형

\* 현금



현재 비중

목표 비중

차이



\---



\### 3. 집중 투자 위험



조건



\* 단일 종목 30% 초과



출력



"집중 투자 위험"



\---



\### 4. 현금 및 파킹 자산 분석



조건



\* 목표 대비 부족 여부



출력



"추가 매수 여력 부족"

또는

"매수 대기 자금 충분"



\---



\### 5. 최근 거래 분석



최근 30일 거래 내역



\* 매수 횟수

\* 매도 횟수

\* 주요 종목



\---



\### 6. AI 행동 제안



예시



\* 리밸런싱 검토

\* 비중 확대 검토

\* 차익실현 검토

\* 추가 적립 투자 검토
